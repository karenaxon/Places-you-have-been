This website lets you create a list of the places where you have been.

## Technologies Used
* C#
* AspNetCore
* MSTest
* Razor
* .Net5

![-----------------------------------------------------](https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/aqua.png)

## Setup/Installation 
* Click on the [repository's](https://github.com/karenaxon/programming-language-suggester.git) link.
* Click on the green "Code" button and copy the repository URL.
* Open your terminal and navigate to the location where you would like to clone the application.
* Use the command _git clone https://github.com/karenaxon/Places-you-have-been.git_ to clone the repository.
* From the top level directory, navigate to the Places folder and use the command _dotnet restore_ to download the packages that the application needs.
* Build the application with the command _dotnet build_.			
* Open the application in your favorite text editor to make changes.

![-----------------------------------------------------](https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/aqua.png)


## To Run Tests 
After following the Setup/Installation steps above:

* Navigate to the Places.Tests folder.
* Run the tests with the command _dotnet test_

![-----------------------------------------------------](https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/aqua.png)

## Using the Application
After following the Setup/Installation steps above: 
  * Navigate to the Places folder 
  * Start the application in a server with the command _dotnet run_.
